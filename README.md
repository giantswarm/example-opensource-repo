# example-opensource-repo

This is a template repository for Giant Swarm Open Source projects

## How to build

### Dependencies

- [github.com/spf13/viper](https://github.com/spf13/viper)

### Building the standard way

```
go get github.com/spf13/viper
go build main.go
```
